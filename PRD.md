1\. PRD (Product Requirements Document)  
**Nome do Projeto:** The Redemption Project (MVP) **Estilo Visual:** Neobrutalism Clean (Bordas marcadas, P\&B predominante, Acentos de cor, Sombras duras).  
1\. Visão do Produto  
Uma plataforma de ensino teológico robusta para jovens e adultos, focada na leitura imersiva e estudo bíblico. O diferencial é a curadoria visual e a experiência de usuário. O conteúdo "Fun Bible Kids" atua como um repositório de recursos premium para pais, acessível via uma área dedicada.  
2\. Estrutura de Conteúdo (Baseada nos PDFs)  
O sistema deve estar preparado para receber dois tipos de dados:  
• **Módulos de Estudo (Texto Rico/Markdown):** Para os E-books "O Plano de Redenção", "Visão Geral VT" e "Visão Geral NT".  
• **Recursos para Download/Visualização (Mídia):** Para os PDFs do Fun Bible Kids (Caligrafia, Caça-Palavras, Desenhos, Histórias).  
3\. Funcionalidades Principais  
• **LMS Minimalista (Learning Management System):**  
    ◦ Barra lateral de navegação dos capítulos (colapsável).  
    ◦ Área de leitura limpa (foco na tipografia).  
    ◦ Checklist de progresso ("Marcar como lido").  
• **Kids Resource Hub:**  
    ◦ Galeria em grid (Neobrutalista).  
    ◦ Filtros por categoria: *Histórias, Atividades, Colorir, Matemática*.  
    ◦ Botão de Download ou "Ler Agora" (para histórias).  
• **Dashboard do Usuário:**  
    ◦ Visão clara de onde parou.  
    ◦ Gamificação sutil (ex: "3 Dias seguidos de estudo").  
4\. Diretrizes de Design (Neobrutalismo Moderno)  
• **Bordas:** Pretas, espessura 2px ou 3px (`border-black border-2`).  
• **Sombras:** Duras, sem desfoque (`shadow-[4px_4px_0px_0px_rgba(0,0,0,1)]`).  
• **Cores:** Fundo Branco/Off-white. Texto Preto Absoluto. Cores de destaque vibrantes (Amarelo Marca-texto, Azul Elétrico, Laranja) usadas com parcimônia.  
• **Tipografia:** Sans-serif robusta (ex: *Inter* ou *Space Grotesk*) para títulos; Serifada moderna para leitura longa (ex: *Merriweather*).
